Sketchfab Model → Cyberpunk 2077 (Full Beginner Guide)

Step‑by‑Step Workflow using Blender + WolvenKit

⚙ Required Software

  • Blender
  • WolvenKit
  • Cyber Engine Tweaks
  • Sketchfab Model (FBX / GLB)

STEP 1 — Import Model into Blender

Delete default cube → File → Import → FBX or GLTF.

STEP 2 — Scale & Apply Transform

Press CTRL + A → Apply All Transforms.

STEP 3 — Attach to Cyberpunk Body Rig

Parent mesh to armature and assign weight to foot bone.

STEP 4 — Export FBX

  • Selected Objects ✓
  • Apply Transform ✓
  • Disable Add Leaf Bones

STEP 5 — Import into WolvenKit

Tools → Import Tool → Add FBX → Import Selected.

STEP 6 — Convert to Wearable Item

Create garment item → assign Feet slot → build mod.
